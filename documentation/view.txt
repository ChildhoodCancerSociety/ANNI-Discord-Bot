COG: view
NAME: view

DESCRIPTION:
This bot saves an organized list of server members as
well as usable data. Such as position in company,
team, team name, join date, etc. This data is used
to run bot features.

USAGE:
-To get a list of all data collected by the bot:
	!view all

-To search or filter data, enter a field to search
followed by a value to search for. For example, to
find all members named 'billy':
	!view name billy

-To get a list of all searchable fields:
	!view fields
