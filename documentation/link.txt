COG: time
NAME: link

DESCRIPTION:
	This command is used to save URL links to the bot. This links can be displayed
	as in alerts. For example, a zoom link can be saved to the bot for easy access.

USAGE:
	To save a new URL to the bot:
		!link save [name] [URL]

	To view saved URLs:
		!link all

	To remove a URL from the bot:
		!link remove [name]
